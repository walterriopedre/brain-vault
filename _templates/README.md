# Templates

Copy from here when starting something new. Don't edit templates in place — duplicate first.

## Available templates

- `runbook.md` — operational procedure template (purpose, prerequisites, steps, rollback, validation).
- `task.md` — task file (frontmatter: id, status, owner, due, source). Drop into `tasks/open/`.
- `meeting-note.md` — meeting notes template. Drop into `meetings/YYYY-MM-DD-<topic>.md`.
- `agents.md` — AGENTS.md starting point for a new vault instance.
- `knowledge-system.md` — knowledge page for a logical system the team owns or operates.
- `knowledge-decision.md` — architectural decision record (ADR).
- `knowledge-solution.md` — resolved incident or problem, indexed by symptom.
- `knowledge-concept.md` — hard-won understanding beyond the standard docs.

Add more as patterns emerge.
