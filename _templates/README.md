# Legacy Claude-Compatible Templates

Copy from here when working with older Claude-oriented workflows. For public-facing starter
templates, prefer `templates/`.

## Available templates

- `runbook.md` — operational procedure template (purpose, prerequisites, steps, rollback, validation).
- `task.md` — task file (frontmatter: id, status, owner, due, source). Drop into `business/tasks/open/`.
- `meeting-note.md` — meeting notes template. Drop into `business/meetings/YYYY-MM-DD-<topic>.md`.
- `agents.md` — AGENTS.md starting point for a new Brain Vault instance.
- `knowledge-system.md` — knowledge page for a logical system, process, or tool.
- `knowledge-decision.md` — architectural decision record (ADR).
- `knowledge-solution.md` — resolved incident or problem, indexed by symptom.
- `knowledge-concept.md` — hard-won understanding beyond the standard docs.

Add more as patterns emerge.
