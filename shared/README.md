# Shared Vault Area

Use this folder for patterns that work in both personal and business vaults.

Examples:

- Naming conventions.
- Reusable checklists.
- Decision-making prompts.
- Common review questions.
- General knowledge that is safe for the intended audience.

Shared material should be audience-safe. If a note contains private personal details or business
confidential material, keep it in the appropriate private area instead.
