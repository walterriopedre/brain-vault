# Business Vault Area

Use this folder for business-mode organization in a private work vault.

Good fits:

- Team operating notes.
- Project briefs.
- Runbooks and procedures.
- Decision records.
- Client-safe context approved for the vault.

Do not use this public template for real company, customer, employee, legal, financial, or
regulated information. Public examples belong in `examples/business/` and must be fake.

Suggested starting points:

- Copy `templates/business/project-brief.md` for a work initiative.
- Copy `templates/business/team-runbook.md` for a repeatable process.
- Store official-system links instead of copying sensitive source records.
