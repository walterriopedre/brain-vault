# AI Usage Area

Use this folder for AI-facing instructions that are not tied to one vendor.

Good fits:

- Preferred AI workflows.
- Prompt patterns.
- Review checklists.
- Tool-use boundaries.
- Notes on what the AI should read before helping.

Claude Code-specific compatibility files remain in `_claude/` and `CLAUDE.md`. General AI
guidance belongs here or in `docs/ai-usage-guide.md`.
