# CLAUDE.md

Claude Code should treat this repository as a public-safe Brain Vault template, not as a
company-only vault.

At the start of a session, read:

@AGENTS.md
@docs/how-this-works.md
@docs/ai-usage-guide.md
@docs/privacy-and-security.md

For setup or customization work, also read:

@INSTALL.md
@docs/customization-guide.md
@docs/migration-from-team-brain.md

For any knowledge query, read `shared/knowledge/index.md` first, then drill into relevant pages.

Memory for this project lives at `_claude/memory/` with index `_claude/MEMORY.md`.
Never save Claude state to the Claude app folder. The vault should remain the portable source
of truth.

Public safety reminder: use fake examples only in this repository. Real private vaults may adapt
these instructions, but this template should not contain real personal, business, customer,
employee, legal, medical, financial, regulated, or secret data.
