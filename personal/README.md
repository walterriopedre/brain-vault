# Personal Vault Area

Use this folder for personal-mode organization in a private clone of Brain Vault.

Good fits:

- Personal projects and goals.
- Learning notes.
- Creative work.
- Household admin.
- Routines and recurring checklists.

Do not put real private personal data in the public template. Public examples belong in
`examples/personal/` and must be fake.

Suggested starting points:

- Copy `templates/personal/personal-project.md` for a project.
- Copy `templates/personal/routine.md` for a recurring habit or checklist.
- Keep reusable lessons in `shared/knowledge/` when they are not private.

Suggested subfolders:

- `personal/projects/` - personal projects.
- `personal/tasks/` - personal next actions.
- `personal/routines/` - recurring routines and checklists.
- `personal/notes/` - general personal notes.
