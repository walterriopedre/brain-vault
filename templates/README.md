# Templates

Copy templates from this folder when starting new notes. Do not edit a template in place unless
you are intentionally improving the starter for future use.

Sections:

- `personal/` - personal projects, routines, learning, life admin.
- `business/` - work projects, runbooks, stakeholder notes, operating records.
- `shared/` - notes, decisions, tasks, and reusable structures that fit either mode.

The legacy `_templates/` folder remains for Claude Code compatibility. Prefer this folder for
public-facing templates.
